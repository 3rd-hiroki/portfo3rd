# 河野浩喜 | ポートフォリオ

このリポジトリは GitHub Pages 用のポートフォリオサイトです。
URL形式：`https://ユーザー名.github.io/portfolio/`

## 公開手順

1. この ZIP を展開し、GitHub の新しいリポジトリ（例：portfolio）を作成。
2. `index.html` と `README.md` をアップロード。
3. リポジトリの Settings → Pages → Source を「Deploy from branch / main / (root)」に設定。
4. 数十秒後、ページが公開されます。
